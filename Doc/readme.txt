【软件说明】
 * 软件名称：   TARKBOT R20系列机器人底层驱动程序 
 * 主要功能：   支持R20系列多款底盘
               包含麦轮款、四轮差速款，阿克曼款、两轮款、履带款

【硬件说明】
 * 硬件平台：   OpenCTR H60 V3.2
 * 处理器：     STM32F407VET6
 * 高速晶振：   8MHz
 * 总线时钟：   系统时钟 = SYCCLK = AHB1 = 168MHz
 * 硬件接口：   参考开源硬件原理图
   
   
 【协议说明】  
 * 串口协议:    塔克通用的X-Protocol协议


【其它说明】
 * 为了获取更好的阅读体验，建议将Tab size设置为4
 * 通过Edit菜单下，Configuration子菜单设置窗口的Editor标签页的“Tab size”选项设置
